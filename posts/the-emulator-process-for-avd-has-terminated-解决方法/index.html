“The emulator process for AVD has terminated”解决方法

2022-12-03 :: kerwinglas
#安卓开发  #android studio  #bugs 

  • 问题描述⌗

    在Device Manager中打算lauch a emulator时失败，报错“The emulator process for AVD has terminated”。

    原因分析⌗

    AVD所在磁盘空间不足。

    The space depends on emulator type. Several are below from my setup (x86_64): Pixel_5_API_31 - 3.6GB, Nexus_7_API_28 - 2.4GB, Nexus_4_API_22 - 5.5GB, Nexus_5X_API_26 - 5.2GB, Nexus_5X_API_28 - 2.7GB, Pixel_API_21 - 5.6GB

    解决方案⌗

    1. 更改AVD的默认路径。参考博客Android 修改AVD路径。

    2. 重启Android Studio。

    Reference⌗

    1. Android Emulator issues in new versions - The emulator process has terminated
    2. Android 修改AVD路径
