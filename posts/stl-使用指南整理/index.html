STRING⌗

STL中的string是一个标准库类，用于表示和处理字符串。以下是STL中string的常用用法：

创建和初始化字符串⌗

可以使用以下方法创建和初始化字符串：

string str1;           // 创建一个空字符串
string str2("hello");  // 创建一个字符串，并将其初始化为"hello"
string str3 = "world"; // 创建一个字符串，并将其初始化为"world"
string str4(5, 'a');   // 创建一个包含5个'a'字符的字符串

访问字符串中的字符⌗

可以使用以下方法访问字符串中的字符：

string str = "hello";
char c1 = str[0];             // 获取字符串的第一个字符'h'
char c2 = str.at(1);          // 获取字符串的第二个字符'e'
char c3 = str.front();        // 获取字符串的第一个字符'h'
char c4 = str.back();         // 获取字符串的最后一个字符'o'

操作字符串⌗

可以使用以下方法对字符串进行操作：

luaCopy codestring str1 = "hello";
string str2 = "world";
string str3 = str1 + str2;  // 将两个字符串连接起来，结果为"helloworld"
str1.append(str2);          // 将str2添加到str1的末尾，结果为"helloworld"
str1.erase(0, 1);           // 删除str1中从位置0开始的1个字符，结果为"ello"
str1.insert(1, "i");        // 在str1的第2个位置插入字符"i"，结果为"hello"

获取字符串的长度⌗

可以使用以下方法获取字符串的长度：

string str = "hello";
int len = str.length();   // 获取字符串的长度，结果为5

比较字符串⌗

可以使用以下方法比较两个字符串：

string str1 = "hello";
string str2 = "world";
if (str1 == str2) {
    // 两个字符串相等
}
if (str1 < str2) {
    // str1小于str2
}
if (str1.compare(str2) == 0) {
    // 两个字符串相等
}

查找字符串⌗

可以使用以下方法在字符串中查找子字符串：

string str = "hello world";
size_t pos = str.find("world");    // 在str中查找"world"，返回其起始位置9
if (pos != string::npos) {
    // 找到了"world"
}

转换字符串⌗

可以使用以下方法将字符串转换为其他类型的值：

string str = "123";
int num = stoi(str);            // 将字符串转换为整数，结果为123
double d = stod("3.14");         // 将字符串转换为浮点数，结果为3.14
long l = stol("123456");         // 将字符串转换为长整数，结果为123456

这些是STL中string的一些常用用法。需要注意的是，string类是可变的，也就是说，可以通过各种操作来改变字符串的内容。

VECTOR⌗

vector 是 STL 中的一个容器，用于存储一组动态分配的对象，具有可变大小和快速随机访问元素的能力。下面是 vector 的详细用法。

头文件和命名空间⌗

使用 vector 需要包含 <vector> 头文件，并使用 std 命名空间。

#include <vector>
using namespace std;

定义 vector⌗

可以通过以下两种方式定义一个 vector：

使用默认构造函数⌗ 

vector<int> vec; // 定义一个空的 int 类型的 vector

使用带有初始值的构造函数⌗ 

vector<int> vec1(10);      // 定义一个包含 10 个 int 类型的元素的 vector，初始值为 0
vector<int> vec2(10, 5);   // 定义一个包含 10 个 int 类型的元素的 vector，初始值为 5
vector<int> vec3 = {1, 2}; // 定义一个包含 2 个 int 类型的元素的 vector，初始值为 1 和 2

插入元素⌗

可以通过以下几种方式向 vector 中插入元素：

使用 push_back() 函数在末尾插入元素⌗ 

vector<int> vec;
vec.push_back(1); // 在末尾插入元素 1
vec.push_back(2); // 在末尾插入元素 2

使用 insert() 函数插入元素⌗ 

vector<int> vec = {1, 2, 3};
vec.insert(vec.begin() + 1, 4); // 在第二个位置插入元素 4

使用 emplace_back() 函数插入元素⌗ 

vector<pair<int, string>> vec;
vec.emplace_back(1, "a"); // 在末尾插入一个 pair<int, string> 类型的元素

访问元素⌗

可以通过下标或迭代器来访问 vector 中的元素。

使用下标访问元素⌗ 

vector<int> vec = {1, 2, 3};
cout << vec[0] << endl; // 输出第一个元素

使用迭代器访问元素⌗ 

vector<int> vec = {1, 2, 3};
auto it = vec.begin(); // 获取迭代器指向第一个元素
cout << *it << endl;   // 输出第一个元素

删除元素⌗

可以通过以下几种方式从 vector 中删除元素：

使用 pop_back() 函数删除末尾元素⌗ 

vector<int> vec = {1, 2, 3};
vec.pop_back(); // 删除末尾元素

使用 erase() 函数删除元素⌗ 

vector<int> vec = {1, 2, 3};
vec.erase(vec.begin() + 1); // 删除第二个元素

unordered_set⌗

unordered_set是C++ STL中的一个容器，用于存储不重复的元素。与其他集合容器不同的是，unordered_set的元素没有顺序限制，因此它的插入和查找操作具有常数级别的时间复杂度。

以下是unordered_set的详细使用说明：

头文件⌗

使用unordered_set需要包含头文件<unordered_set>。

定义⌗

unordered_set的定义方式与其他STL容器类似，可以使用下面的语法：

#include <unordered_set>
using namespace std;
unordered_set<int> mySet;

其中，unordered_set表示存储整型元素的unordered_set，mySet是unordered_set对象的名称。

插入元素⌗

unordered_set提供了insert()函数用于插入元素，示例代码如下：

mySet.insert(1);
mySet.insert(2);
mySet.insert(3);

上述代码插入了三个整型元素1、2、3。

删除元素⌗

unordered_set提供了erase()函数用于删除元素，示例代码如下：

mySet.erase(2);

上述代码删除了元素2。

查找元素⌗

unordered_set提供了find()函数用于查找元素，示例代码如下：

if (mySet.find(1) != mySet.end()) {
    cout << "Found!" << endl;
}

上述代码查找元素1，如果找到了就输出"Found!"。

遍历元素⌗

unordered_set的元素没有顺序限制，因此不能使用下标操作符[]来访问元素。可以使用迭代器来遍历元素，示例代码如下：

for (auto it = mySet.begin(); it != mySet.end(); it++) {
    cout << *it << endl;
}

上述代码遍历unordered_set中的所有元素，并输出它们的值。

其他操作⌗

unordered_set还提供了一些其他操作，例如：

  • size()：返回unordered_set中元素的个数。
  • clear()：清空unordered_set中的所有元素。
  • empty()：判断unordered_set是否为空。
  • emplace()：以构造函数参数的方式插入元素，效率比 insert 方法高。

以上就是unordered_set的详细使用说明。需要注意的是，unordered_set是C++11引入的新特性，因此在编译时需要使用支持C++11的编译器。
