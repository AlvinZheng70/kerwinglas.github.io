Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

Example :

Input: nums = [1,2,3,1]
Output: true

Summary⌗

这道题比较简单，最容易想到的思路是排序，即先将数据进行排序，再遍历一遍数组查看是否有重复元素。时间复杂度是O(nlogn)，时间性能较差。还有一种思路是使用数据结构哈希表，可以利用STL库中的unordered_set，时间复杂度降为O(n)。这个数据结构，从逻辑上看可以简单看作一个不容纳重复元素的集合，从实现上看是利用hash映射存值，从而加快插入和查找。具体思路是遍历一遍数组，逐个加入元素至集合中，加入前判断其是否已经存在于集合中。注意unordered_set的具体用法。
